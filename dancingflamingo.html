Dancing Flamingo

iOS Game App 2014

This is my first iOS Game Application after graduationg iOS programming bootcamp in SF in 2014. I draw, designed and hand-coded this Puzzle Game by myself from scratch to deepen my knowledge of Objective-C and Cocos2D-x(2D Games framework) on X-code.

Skillset: UI Design, Game Design, Illustration, Programming & Monetization
Tools: Sketch, Photohsop, Illustorator, X-code, Cocos2D-X, Atom, Git

Features: Left Swipe, Right Swip , Up Swipe, Down Swipe, Tap, Sound
